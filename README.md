# 🚀 My Portfolio
Welcome to my personal portfolio! This project is a showcase of my skills and experience as a Mobile App Developer, built using Flutter. It features a clean and modern design, responsive layout, and smooth animations to provide an engaging user experience.
## ✨ Key Features

- **📱 Fully Responsive**: Seamless experience across Mobile, Tablet, and Desktop using `responsive_framework`.
- **🎨 Rich Aesthetics**: Elegant UI with custom themes, `google_fonts`, and high-quality vector graphics (`flutter_svg`).
- **🎬 Advanced Animations**: Dynamic interactions powered by `flutter_animate` and `lottie` for a premium feel.
- **🗺️ Robust Routing**: Clean and scalable navigation using `go_router`.
- **🌓 State Management**: Efficient data flow and state handling with `provider`.
- **📞 Direct Contact**: Integrated social links and contact capabilities using `url_launcher` and Font Awesome.

---

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **State Management**: [Provider](https://pub.dev/packages/provider)
- **Navigation**: [GoRouter](https://pub.dev/packages/go_router)
- **UI & Layout**: [Responsive Framework](https://pub.dev/packages/responsive_framework), [Google Fonts](https://pub.dev/packages/google_fonts)
- **Animations**: [Flutter Animate](https://pub.dev/packages/flutter_animate), [Lottie](https://pub.dev/packages/lottie)
- **Icons**: [Font Awesome Flutter](https://pub.dev/packages/font_awesome_flutter), [Cupertino Icons](https://pub.dev/packages/cupertino_icons)

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (v3.38 or later)
- Android Studio / VS Code with Flutter extension

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/javed-khan-developer/my_portfolio.git
   
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run -d chrome  # For Web
   # OR
   flutter run           # For Mobile/Desktop
   ```
